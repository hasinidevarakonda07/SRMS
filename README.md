
# Student Record Management System (SRMS)

## Overview

The Student Record Management System (SRMS) is a console-based application developed in C for managing student records efficiently. The system provides secure role-based access through a login mechanism and allows administrators, staff members, and guests to access features according to their permissions.

## Features

* Secure login authentication with hidden password input
* Role-based access control (Admin, Staff, Guest)
* Add new student records
* View all student records
* Search students by roll number or name
* Update existing student information
* Delete student records
* File-based data storage using text files

## Technologies Used

* C Programming Language
* File Handling
* Structures
* Functions
* Role-Based Authentication

## Project Structure

* `students.txt` – Stores student records
* `credentials.txt` – Stores login credentials and user roles
* `SRMS.c` – Main source code

## Learning Outcomes

This project demonstrates the implementation of file handling, structures, authentication systems, and CRUD (Create, Read, Update, Delete) operations in C. It provides a practical solution for managing student data while reinforcing fundamental programming concepts.
